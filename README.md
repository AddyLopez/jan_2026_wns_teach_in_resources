# WNS Teach-In Resources (Jan. 2026)

## Description

A list of resources for a teach-in hosted and compiled by Workers for a New Southwest (WNS). WNS is a local grassroots organization based in Albuquerque, New Mexico. To find out more, visit WNS's Instagram account here: [https://www.instagram.com/workersforanewsouthwest/?hl=en](https://www.instagram.com/workersforanewsouthwest/?hl=en).
