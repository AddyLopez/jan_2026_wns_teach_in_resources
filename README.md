# WNS Teach-In Resources (Jan. 2026)

## Description

A list of resources compiled by Workers for a New Southwest (WNS) for a teach-in hosted at the Albuquerque Center for [Peace and Justice](https://www.abqpeaceandjustice.org/mission-statement). WNS is a local grassroots organization based in Albuquerque, New Mexico. To find out more, visit WNS's Instagram account here: [https://www.instagram.com/workersforanewsouthwest/?hl=en](https://www.instagram.com/workersforanewsouthwest/?hl=en). This website was coded _pro bono_.
